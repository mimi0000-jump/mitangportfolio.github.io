Project 1: AI Driven Synthesizing System

This project is an AI-driven synthesizer system built with Max/MSP and integrated with ChatGPT. It converts text into sound and allows users to adjust synthesizer presets, drum patterns, etc. Most of the parameters are mapped based on emotional coordinates (valence and arousal), with some calculated internally and others generated by ChatGPT. - Specific Role: System Engineer (API, JavaScript, MaxMSP), Producer, Sound Designer. - Technologies Used: MAX/MSP, AI Model API, JavaScript.
